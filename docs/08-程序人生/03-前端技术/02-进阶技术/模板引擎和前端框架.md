# 模板引擎

模板引擎（这里特指用于Web开发的模板引擎），是为了使用户界面与业务数据（内容）分离而产生的，它可以生成特定格式的文档，用于网站的模板引擎就会生成一个标准的html文档；

常见：Thymeleaf 、FreeMarker、Velocity

# 模板引擎原理

- 首先：客户端浏览器发起HTTP请求到servlet中；
- servlet 调用模板引擎组织 **模板（静态不动的资源)+数据(动态的资源）**，将两者拼接为一个HTML字符串；
- 将该HTML字符串返回给服务端；
